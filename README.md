# Claude Resources

A curated collection of resources for Claude Code users, including MCP servers and Claude Skills/Hooks.

## Contents

### [MCP Servers](mcp-servers.md)

A comprehensive list of MCP (Model Context Protocol) servers organized by category:

- Development & Debugging
- Documentation & Research
- Framework-Specific (Nuxt, Svelte, Flutter, Shadcn)
- Cloud & Infrastructure
- Google Services
- Database & Backend
- Payments & Business
- Project Management & Productivity
- Data & Web Scraping
- Security
- Voice & Communication
- Memory & Context
- Workflow & Orchestration

Each server entry includes installation instructions for both CLI and JSON configuration.

### [Claude Skills & Hooks](claude-skills.md)

Community-built Claude Skills and Hooks including:

- Essential Skills (Superpowers, Skill Seekers)
- Developer Workflow (TDD, Debugging, Git Worktrees, Testing)
- Research & Knowledge (Content extraction, Brainstorming)
- Productivity & Automation
- Claude Code Hooks (TypeScript, Python, PHP implementations)
